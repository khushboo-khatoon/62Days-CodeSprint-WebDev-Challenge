# AI Interview Prep

AI-powered interview preparation platform built using MongoDB, Express.js, React.js, and Node.js.

## Features

- User Authentication
- AI Interview Questions
- Mock Interviews
- Performance Tracking
- Progress Dashboard
- Interview History
- Reports & Analytics

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- Gemini AI

## Setup

### Client

npm install
npm run dev

### Server

npm install
npm run dev
