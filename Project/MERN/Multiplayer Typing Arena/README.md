# Realtime Multiplayer Typing Arena

Educational MERN project for issue **#252**.

## Features
- Auth + player profiles
- Socket.io race rooms
- Live WPM / accuracy broadcast
- Basic anti-cheat timing validation
- Match replay stored in MongoDB
- Post-match analytics

## Run
```bash
cd backend && cp .env.example .env && npm install && npm run dev
cd frontend && npm install && npm run dev
```

## Tech Stack
MongoDB · Express · React (Vite) · Node.js · Socket.io
