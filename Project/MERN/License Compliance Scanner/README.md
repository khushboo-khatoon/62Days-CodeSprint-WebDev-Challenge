# Open-Source License Compliance Scanner

Educational MERN project for issue **#253**.

## Features
- Paste public GitHub repo URL and scan
- Fetch package.json via public APIs
- License detection + copyleft/unknown risk flags
- Scan history in MongoDB
- CSV / printable report export
- Educational risk explanations

## Run
```bash
cd backend && cp .env.example .env && npm install && npm run dev
cd frontend && npm install && npm run dev
```
