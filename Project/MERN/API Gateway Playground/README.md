# API Gateway Playground

Educational MERN project for issue **#249**.

## Features
- Mock microservices & routes
- JWT / API-key policy layers
- Latency injection + rate limiting
- Proxy/mock upstream responses
- Request logs in MongoDB + React console

## Run

```bash
cd backend && cp .env.example .env && npm install && npm run dev
cd frontend && npm install && npm run dev
```

## Tech Stack
MongoDB · Express · React (Vite) · Node.js
