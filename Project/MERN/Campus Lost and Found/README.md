# Campus Lost & Found

Educational MERN project for issue **#250**.

## Features
- JWT + admin moderation
- Lost/found CRUD with image uploads (multer)
- Tag search + OpenStreetMap pins (Leaflet)
- Claim workflow + in-app messaging
- Educational image match score helper

## Run

```bash
cd backend && cp .env.example .env && npm install && npm run dev
cd frontend && npm install && npm run dev
```

## Tech Stack
MongoDB · Express · React (Vite) · Node.js

Uses **multer** + **Leaflet** CDN.
