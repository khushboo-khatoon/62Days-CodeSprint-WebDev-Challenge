## ✨ Features

- 🎯 Two-player gameplay (Player X vs Player O)
- 🏆 Automatic winner detection
- 🤝 Draw game detection
- 🔄 Restart game anytime
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 🎮 How to Play

1. Player **O** makes the first move.
2. Players take turns placing their marks on the board.
3. The first player to align 3 marks horizontally, vertically, or diagonally wins.
4. If all cells are filled and no one wins, the game ends in a draw.
5. Click the **Restart** button to play again.
