## ✨ Features

*   **User-Friendly Interface:** Just enter a GitHub username to get started.
*   **Key Statistics:** Displays important metrics like:
    *   Followers and Following count
    *   Number of Public Repositories
    *   Number of Public Gists
*   **Direct Profile Link:** Includes a link to the user's GitHub profile.
*   **Dynamic & Responsive:** Built with modern web technologies for a seamless experience on any device.

---

## 🚀 How to Use

1.  Clone or download this repository.
2.  Open the `index.html` file in your web browser.
3.  Enter a valid GitHub username into the input field.
4.  Click the "Get Stats" button.
5.  The user's profile stats will be displayed on the screen.

---

## 🛠️ Technologies Used

*   **HTML5:** For the basic structure of the web page.
*   **CSS3:** For styling and layout.
*   **JavaScript (ES6+):** For fetching data from the GitHub API and dynamically updating the page.
*   **GitHub API:** The source of all the profile data.
