# Amazon Clone 🛒

A frontend Amazon clone project built to replicate the look and feel of Amazon's shopping website. This project focuses on creating a familiar e-commerce interface with product sections, navigation, banners, and shopping-related UI components.

> Note: This project is currently **not fully responsive** and is optimized mainly for desktop screens.

## 📌 Project Overview

The Amazon Clone is a web application that recreates the Amazon shopping experience using modern frontend technologies. It includes a homepage layout with product cards, navigation menus, categories, and other essential e-commerce UI elements.

This project was created for learning purposes to improve frontend development skills and understand how large-scale e-commerce websites structure their interfaces.

## ✨ Features

- ✅ Amazon-inspired homepage design
- ✅ Navigation bar with search functionality UI
- ✅ Product cards and product sections
- ✅ Hero banner section
- ✅ Category sections
- ✅ Footer section
- ✅ Clean and reusable components
- ✅ Modern UI implementation

## 🚧 Current Limitations

- ❌ Not responsive for mobile devices
- ❌ No backend integration
- ❌ No authentication system
- ❌ No real payment functionality
- ❌ Product data is static

## 🛠️ Tech Stack

- HTML5
- CSS3
