## ✨ Features

- **Single-file & offline**: Works by opening the HTML file in your browser (no install / no build).
- **Fill in your details**: Name, tagline/title, about, skills, GitHub username, X, and LinkedIn.
- **Live Markdown preview**: Updates instantly as you type.
- **GitHub stats (optional)**: If you enter a GitHub username, the app fetches repo / star / fork counts from GitHub’s public API.
- **Copy & Download**:
  - Copy the generated Markdown to clipboard
  - Download it as `README.md`

---

## 🚀 How to Run

1. Download/clone this repository.

2. Open `github-profile-readme-generator.html` in any modern browser (Chrome, Firefox, Edge, Safari).

That’s it—start generating your profile README.

---

## Notes

- **GitHub stats require network access** and may be **rate limited**.
- The live preview uses the `marked` library loaded from a CDN.
